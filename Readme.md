# 🦒 Бот-викторина «Твоё тотемное животное» - Московский зоопарк

Telegram-бот для популяризации программы опеки [Московского зоопарка](https://moscowzoo.ru/).  
Пользователь проходит короткую викторину из 10 вопросов и узнаёт, какое животное зоопарка является его тотемным, а также - как взять это животное под опеку.

---

## ✨ Функциональность

| Возможность | Описание |

| 📲 Поделиться результатом | Нативная кнопка Telegram с предзаполненным текстом и ссылкой на бота |
| 👍 Отзывы | Оценка бота, уточняющие варианты и свободный текст отзыва |
| 🔔 Уведомления | Администратор получает результат каждого прохождения с кнопкой «Написать пользователю» |

| ❓ /help | Команда-справочник по всем возможностям бота |
| 🛡️ Fallback-хэндлеры | Бот корректно реагирует на любой неожиданный ввод |

## 🛠 Технологии

- **Python 3.10+**
- **[aiogram 3.13](https://docs.aiogram.dev/)** - асинхронный фреймворк для Telegram Bot API
- **[python-dotenv](https://pypi.org/project/python-dotenv/)** - управление переменными окружения
- **FSM** (Finite State Machine) - управление состояниями диалога

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/Recwayer/ZooMoskowBot.git
cd ZooMoskowBot
```

### 2. Создание виртуального окружения

```bash
python -m venv venv

# Linux / macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 4. Настройка переменных окружения

```bash
cp .env.example .env
```

Откройте `.env` и заполните:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_chat_id
```

- **`BOT_TOKEN`** - получите у [@BotFather](https://t.me/BotFather) в Telegram
- **`ADMIN_ID`** - узнайте свой chat_id через [@userinfobot](https://t.me/userinfobot) (опционально: если не указать, уведомления просто не будут отправляться)

### 5. Запуск бота

```bash
python bot.py
```

## 🧮 Алгоритм подбора животного

Каждый вариант ответа содержит словарь `animal_points` - он может начислять очки сразу нескольким животным:

```python
{"text": "Яркость и харизма", "animal_points": {"blue_macaw": 3, "red_flamingo": 2, "toco_toucan": 1}}
```
После 10 вопросов победителем становится животное с максимальной суммой очков. При ничьей или нулевом результате - показывается Малый подорлик (животное по умолчанию).

## 🔔 Уведомления администратора

После каждого прохождения викторины на `ADMIN_ID` отправляется сообщение:

- Имя и username пользователя
- Telegram ID
- Время прохождения
- Тотемное животное с emoji и ссылкой
- Кнопка «Написать пользователю»

Отзывы пользователей также пересылаются администратору с указанием типа отзыва.

## 💬 Механизм сбора отзывов

После результата пользователь видит кнопки оценки:

- **👍 Классный бот!** - позитивный отзыв, уведомление администратору
- **👎 Есть замечания** - открывает меню с вариантами:
  - ❓ Вопросы не понравились
  - 🎯 Результат не подходит
  - 🐛 Бот работал с ошибками
  - ✏️ Написать отзыв текстом (FSM-состояние для свободного ввода)

## 🔒 Безопасность и конфиденциальность

- Токен бота и ID администратора хранятся в `.env`, который добавлен в `.gitignore`
- Бот собирает только минимально необходимые данные: имя, username и Telegram ID
- Никакие данные не сохраняются в базе данных - используется только оперативная память (MemoryStorage)
- Данные каждой сессии удаляются после завершения викторины (`await state.clear()`)

---

## 📈 Масштабируемость и мониторинг

- Бот использует асинхронную архитектуру (`asyncio` + `aiogram`), что обеспечивает высокую пропускную способность
- Все ключевые события логируются через стандартный модуль `logging`
- Для production-деплоя рекомендуется заменить `MemoryStorage` на `RedisStorage` (aiogram поддерживает из коробки)
- Ошибки загрузки фото и отправки уведомлений обрабатываются через `try/except` с записью в лог
